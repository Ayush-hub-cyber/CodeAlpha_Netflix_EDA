# Netflix Exploratory Data Analysis

## Project Overview
This project analyzes the Netflix Titles Dataset using Python, Pandas, and Matplotlib.

## Objectives
- Understand dataset structure
- Identify missing values
- Analyze Movies vs TV Shows
- Find top content-producing countries
- Study release trends over time

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Key Findings
- Netflix contains more Movies than TV Shows.
- United States contributes the highest amount of content.
- Content production increased significantly after 2015.

## Visualizations
1. Movies vs TV Shows
2. Top 10 Countries
3. Release Trend Analysis
